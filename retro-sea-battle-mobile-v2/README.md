# Retro Sea Battle Mobile V2

Компактная версия для телефона: игровой экран уменьшен, чтобы поле чаще помещалось целиком.

## Ссылка на игру

Ожидаемый путь GitHub Pages:
`https://dmitraba.github.io/RAG_local/retro-sea-battle-mobile-v2/`

## Что изменено

- уменьшен игровой контейнер на узких экранах;
- уменьшены отступы и кнопки;
- компактнее HUD и вспомогательные блоки;
- мобильное управление сохранено.
