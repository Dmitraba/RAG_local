# Retro Sea Battle

Небольшой браузерный прототип игры в духе советского игрового автомата "Морской бой".

## Что есть в текущей версии

- ретро-интерфейс с зеленым экраном;
- вертикальное управление прицелом;
- торпеды и движущиеся корабли-цели;
- счет, промахи, таймер и завершение раунда;
- увеличенная длительность игры;
- встроенные ретро-звуки без внешних файлов;
- быстрый рестарт.

## Управление с клавиатуры

- `W` / `S` двигают прицел вверх и вниз
- `Стрелка вверх` / `Стрелка вниз` делают то же самое
- `Пробел` выпускает торпеду
- `Enter` запускает раунд или перезапускает игру

## Управление на телефоне

- кнопка `Старт / рестарт` запускает раунд;
- переключатель `Кнопки / Палец` меняет способ управления;
- кнопки `Вверх` и `Вниз` двигают прицел;
- в режиме `Палец` прицел двигается касанием и перетаскиванием по полю;
- кнопка `Огонь` стреляет;
- играть можно без клавиатуры.

## Файлы

- `index.html` - структура страницы
- `styles.css` - визуальный стиль интерфейса
- `game.js` - логика игры и синтез звуков
- `ЛОГ.md` - рабочий журнал проекта
